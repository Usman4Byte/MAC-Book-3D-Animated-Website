<div style="font-family: Arial, sans-serif; line-height: 1.6; max-width: 900px;">

  <h1 style="font-weight: 700;">
    Apple MacBook Pro Landing Page <span style="font-weight: 400;">(3D & Animation)</span>
  </h1>

  <p>
    Designed and developed a high-fidelity <strong>Apple MacBook Pro landing page</strong>
    focused on immersive visuals and smooth user experience. The project features
    interactive <strong>3D MacBook models</strong> and
    <strong>scroll-based animations</strong>, inspired by modern product-driven web design.
  </p>

  <h2 style="margin-top: 30px;">Key Features</h2>
  <ul>
    <li>Real-time rendered <strong>3D MacBook models</strong></li>
    <li>Smooth <strong>GSAP animations</strong> and transitions</li>
    <li>Scroll-based interactions for product storytelling</li>
    <li>Fully responsive layout across devices</li>
    <li>Clean, modern, Apple-inspired UI</li>
  </ul>

  <h2 style="margin-top: 30px;">Tech Stack</h2>
  <ul>
    <li><strong>React.js</strong> (Vite)</li>
    <li><strong>Three.js</strong> with react-three-fiber & react-three-drei</li>
    <li><strong>GSAP</strong> & gsap-react for animations</li>
    <li><strong>Tailwind CSS</strong> for styling</li>
    <li><strong>Zustand</strong> for state management</li>
  </ul>
  
</div>
